# RF-3DGS
 RF-3DGS (Radio-Frequency 3D Gaussian Splatting) is an open-source framework designed for simulating and reconstructing radio radiance field. For more details, please check our paper: [RF-3DGS: Wireless Channel Modeling with Radio Radiance Field and 3D Gaussian Splatting](https://arxiv.org/abs/2411.19420).
